# 📊 Tanvi Vanani — Data Analyst Portfolio

> A modern, animated portfolio built with vanilla HTML, CSS & JS. No frameworks. GitHub Pages ready.

---

## ✨ Features

- **Animated Neural Network** — live canvas background representing data connections  
- **Terminal Typing Effect** — techy hero section with typewriter animation  
- **Animated Skill Bars** — progress bars fill on scroll (Power BI, SQL, Python, Excel…)  
- **Counter Animations** — stats count up as they enter the viewport  
- **Timeline Experience** — colour-coded work history (full-time / internship / education)  
- **Project Cards** — icon-based cards with tech tags  
- **Certifications Section** — badge-style credential display  
- **Responsive** — mobile-first layout with hamburger nav  
- **Amber/Violet Dark Theme** — Space Mono + Syne typefaces

## 🚀 Deploy to GitHub Pages

1. **Create a new repository** on GitHub (e.g., `tanvi-portfolio` or `tanvivanani25.github.io`)
2. **Push all files** to `main`:
   ```bash
   git init
   git add .
   git commit -m "initial portfolio"
   git remote add origin https://github.com/tanvivanani25/<repo-name>.git
   git push -u origin main
   ```
3. Go to **Settings → Pages → Branch: main → / (root)** → Save  
4. Your site will be live at `https://tanvivanani25.github.io/<repo-name>/`

## 🛠 How to Customise

| What to change | Where |
|---|---|
| Name, bio, contact info | `index.html` |
| Accent colours | `:root` in `style.css` (`--amber`, `--violet`) |
| Skill percentages | `data-width` attributes in `index.html` |
| Add new projects | `#projects` section in `index.html` |
| Neural network density | `14000` divisor in `script.js` |
